uitleg over de taken van de processor

De processor zit in elke computer, telefoon, rekenmachine. Zonder de processor start de computer ook niet op. De processor bepaalt de snelheid van de computer. Hoe beter de processor hoe sneller de computer is: hoe meer berekeningen en aansturingen per seconde verwerkt kan worden hoe sneller de computer is.

Maar er zijn aantal redenen/ factoren waarop de snelheid van de processor wordt bepaald. 1ste: Aantal processorkernen 2e:Kloksnekheid 3e: Cache geheugen 4e: Slimme technieken(turbo boost, hyper threading)

1ste: aantal processorkernen: De aantal processorkernen worden bepaalt hoeveel taken de proseccor tegerlijkertijd kan verwerken.

2e: kloksnelheid: De kloksnelheid van een processor is de snelheid waarmee de processor de instructies kan uitvoeren. Ook dekloksneheid is de verwerkingssnelheid van een processor de snelheid wordt gekozen door de klokchip die geeft steeds het zelfde aantal seconde een puls die gaat naar de processor alle verwerkingsstappen gelijk te laten lopen. Het wordt gemeten in hertz(Hz). Een hogere kloksnelheid betekent over het algemeen dat de processor sneller kan werken.

3e: Cache geheugen: Cache geheugen is een snel, klein en tijdelijk geheugen en dat vindt plaats dichtbij de processor. De cache geheugen wordt gebruikt om veelgebrikte gegevens en instructies op te slaan, dat doen ze, omdat de processor er snel toegang tot geeft. Hierdoor kan de processor handiger wrken en sneller gegevens ophalen dan wanneer het alleen afhankelijk zou zijn van het hoofdgeheugen.

Slimme technieken (turbo boost, hyper threading)

turbo boostL

Turbo boost is een technologie die gebruikt wordt door sommige processors. Het stelt de processor in om tijdelijk te werken op een hogere kloksnelheid dan de standaardkloksnelheid. Dit zorgt voor verbeterede prestatie bij uitvoering van taken.
